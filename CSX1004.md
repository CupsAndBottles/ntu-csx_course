# 網頁互動程式設計 (Interactive Web Programming)

#### 課程介紹

JavaScript是網頁前端開發的程式核心，主要是幫助實現頁面的互動及一些特效的實現。本課程將教授JavaScript的基本語法，學習用JavaScript提高豐富網頁互動效果。
